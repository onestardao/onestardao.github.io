# WFGY

WFGY is a verification first reasoning ecosystem for LLMs.  
One architecture, different depths. Open Source under the MIT License.

---

## Entry points

- **WFGY 3.0 · Frontier Engine**  
  TXT based stress testing and long horizon reasoning.  
  [Singularity Demo](https://github.com/onestardao/WFGY/blob/main/TensionUniverse/EventHorizon/README.md)

- **WFGY 2.0 · Engineering Engine**  
  Production oriented components for RAG stacks, vector stores, and deployment debugging.  
  [Core Engine](https://github.com/onestardao/WFGY/blob/main/core/README.md)

- **WFGY 1.0 · Foundations**  
  Math, formulas, and the original theory track.  
  [Legacy Theory](https://github.com/onestardao/WFGY/blob/main/legacy/README.md)

- **RAG 16 Problem Map**  
  A compact failure mode checklist and fix map for broken RAG and agent pipelines.  
  [16 Problem Map](https://github.com/onestardao/WFGY/blob/main/ProblemMap/README.md)

- **Global Debug Card**  
  Image as protocol for the 16 Problem Map, poster plus Q E P A context for fast triage.  
  [Global Debug Card](https://github.com/onestardao/WFGY/blob/main/ProblemMap/wfgy-rag-16-problem-map-global-debug-card.md)
